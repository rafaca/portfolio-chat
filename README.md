# Portfolio Chat

AI-powered portfolio assistant for Rafa Castello.

**Live:** [portfolio-chat-swart.vercel.app](https://portfolio-chat-swart.vercel.app)

## Tech Stack

- Next.js 15 (App Router, Turbopack)
- React 19
- Anthropic Claude API
- Tailwind CSS 4
- TypeScript

## Setup

```bash
pnpm install
cp .env.local.example .env.local
# Add your ANTHROPIC_API_KEY and SITE_PASSWORD
pnpm dev
```

## Deployment

Deployed on Vercel. Pushes to `main` auto-deploy.
